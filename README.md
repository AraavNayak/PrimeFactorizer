# primeFactorizer

This program outputs the prime factorization (with exponents) of a number given as input.

<br>
Sample inputs:
<pre>
1536
</pre>

<br>
Sample outputs:
<pre>
Factors: 1,2,3,4,6,8,12,16,24,32,48,64,96,128,192,256,384,512,768,1536
Factorization terminated;
Is prime: false
Is composite: true
20 factors.
2^9 x 3
</pre>
